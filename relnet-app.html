    📥 Facebook Data Import

    How to Export
    Friends List
    Message Data

    Step 1 — Request your Facebook data

    Go to Facebook → Settings → Your Facebook Information → Download Your Information.
    Select: Friends and Messages. Format: JSON. Date range: All time. Click "Create File".

    Step 2 — Wait & Download

    Facebook will notify you (usually 1–24 hours). Download the ZIP file. Extract it. You'll find JSON files inside friends_and_followers/ and messages/inbox/.

    Step 3 — Import here

    Use the tabs above to import your friends.json (contacts) and message data (to infer relationship strength from conversation frequency).

    Import friends.json

    From your Facebook export, open friends_and_followers/friends.json. Paste the entire JSON content below.

    Import message data → infer relationship strength

    Each conversation folder contains message_1.json. Paste one or more. Relnet counts messages exchanged per person and uses that to set relationship strength (more messages = stronger tie).

    Format: Facebook's messages/inbox/[person]/message_1.json
