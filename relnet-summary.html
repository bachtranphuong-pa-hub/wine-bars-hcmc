Overview

What is Relnet?

Relnet is a personal relationship intelligence tool — a knowledge graph that maps your contacts as nodes and the relationships between them as edges. Think of it as a private CRM meets LinkedIn graph, but fully under your control, running locally, with no data leaving your device.

The core insight: most people have hundreds of contacts but no visual map of how those people connect to each other. Relnet makes those invisible connections visible — revealing clusters, brokers, and relationship gaps that are hard to see in a flat contact list.

Core Use Cases

  • Network mapping — visualize who knows whom, relationship strength, and cluster affiliations
  • Introduction planning — find the shortest path between you and a target person via mutual connections
  • Contact enrichment — tag contacts by industry, location, relationship type, and social handles
  • Photo-based recognition — import photos, detect faces, and auto-infer co-occurrence relationships from who appears together
  • Relationship health — track last interaction and relationship strength decay over time

Core Logic

Key Algorithms

1. Force-Directed Graph Layout (D3)

// graphLayout.js — D3 simulation config
const createSimulation = (nodes, edges) => {
  return d3.forceSimulation(nodes)
    .force('link', d3.forceLink(edges)
      .id(d => d.id)
      .distance(d => 120 / d.strength)   // stronger = closer
    )
    .force('charge', d3.forceManyBody().strength(-400))
    .force('cluster', forceCluster())    // pull same-tag nodes together
    .force('center', d3.forceCenter(0, 0));
};

// Custom cluster force — pulls nodes with matching tags toward centroid
function forceCluster() {
  return (alpha) => {
    nodes.forEach(node => {
      const centroid = clusterCentroids[node.primaryTag];
      if (centroid) {
        node.vx += (centroid.x - node.x) * 0.05 * alpha;
        node.vy += (centroid.y - node.y) * 0.05 * alpha;
      }
    });
  };
}

2. Face Co-occurrence → Infer Relationships

// faceRecognition.js — after batch photo import
async function inferRelationshipsFromPhotos(photos) {
  const coOccurrences = {};

  for (const photo of photos) {
    const detections = await detectAllFaces(photo);
    const matched = detections.map(matchToContact);

    // Every pair in this photo = co-occurrence event
    for (let i = 0; i < matched.length; i++) {
      for (let j = i + 1; j < matched.length; j++) {
        const key = pairKey(matched[i].id, matched[j].id);
        coOccurrences[key] = (coOccurrences[key] || 0) + 1;
      }
    }
  }

  // Convert counts → edge strength (1-5 scale, log-normalized)
  return Object.entries(coOccurrences).map(([key, count]) => ({
    ...parsePairKey(key),
    strength: Math.min(5, Math.ceil(Math.log2(count + 1))),
    type: 'inferred',
    source: 'photo_cooccurrence'
  }));
}
